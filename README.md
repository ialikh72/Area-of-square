# Square Area Calculator (Python)

## Description

This is a simple Python program that calculates the **area of a square**.
The user enters the length of one side of the square, and the program calculates the area using the formula:

Area = side × side

## Code

```python
side = int(input("Enter the side of the square: "))
area = side * side
print("area of square is:", area)
```

## How It Works

1. The program asks the user to enter the **side length of the square**.
2. The value is converted into an integer using `int()`.
3. The area is calculated by multiplying the side by itself (`side * side`).
4. The result is displayed using `print()`.

## Example Run

```
Enter the side of the square: 5
area of square is: 25
```
